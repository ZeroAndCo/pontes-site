# Fonts

This folder stores self-hosted font files used by the site.

## Bugaki

- Files expected by `css/fonts.css`:
  - `Bugaki.woff2` (preferred)
  - `Bugaki.woff` (fallback)
- Optional variants (add and reference in `fonts.css`):
  - `Bugaki-Bold.woff2` / `Bugaki-Bold.woff` (font-weight: 700)
  - `Bugaki-Italic.woff2` / `Bugaki-Italic.woff` (font-style: italic)

## Licensing

- Ensure you have proper licensing/rights to use and distribute Bugaki.
- Store license text or source attribution here if required.

## Paths

- CSS expects files at `../fonts/*.woff2` and `../fonts/*.woff` relative to `css/fonts.css`.
- GitHub Pages serves these via relative paths; no root `/` prefixes are used.
